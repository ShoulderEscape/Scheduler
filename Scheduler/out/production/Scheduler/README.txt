1. Run the program
2. Choose which sorting method that you want to use
3. If you pick round robin (RR), it will ask you which time quanta that you want to use, that is the time between when you change process
4. You get your results

If you want to change the values that you input, go to input.txt, where the first value is the id of the value, the second is the arrival time, and the third is the burst time.
After that there is a new input, with other ids, arrival times, and burst times
